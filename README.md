# Nontoon-Metallic-smoothness-Module
module for the nontoon shader that enables metallic and smoothness mapping

requires nontoon to work (duh)
https://github.com/lilxyzw/NonToon

set your "roughness" and "use roughness in normal map" to match the below option as this allows the module to override the roughness in the shader
